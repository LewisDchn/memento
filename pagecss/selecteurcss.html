Les sélecteurs d'attributs

Lorsque l’on veut appliquer une propriété CSS à un élément HTML il faut cibler cet élément à l’aide d’un sélecteur.
Le sélecteur se réfère généralement à une balise HTML (« nomBalise »), à une classe(« .nomClass ») ou à un id (« #nomId »).
Il existe aussi le selecteur Universel qui applique la règle à toute les élément de la page HTML (« * »).
Il est possible de groupé plusieurs sélecteurs séparer par une virgule, ils auront alors la même définition de style.
